# Understanding human decision-making in repeated social interaction

This is the repository for the code accompanying the report ``understanding human decision-making in repeated social interaction'' as a course project for 9.660 Computational Cognitive Science by Professor Joshua Tenenbaum at MIT. The code is developed by Xinling Li (xinli831@mit.edu).

## Files
- `IPD_BSL.ipynb`: Jupyter notebook containing the implementation of the Bayesian State Learning (BSL) model for the repeated prisoner's dilemma.
- `BayesianMind.ipynb`: Jupyter notebook containing the implementation of the Bayesian belief model for the repeated prisoner's dilemma.

The author does not have permission to share the dataset used in the report. Please contact the course TA at <a_lamba@mit.edu> for access to the dataset.
